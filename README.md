# 斗鱼鱼吧自动签到辅助工具

这是一款 Chrome 浏览器扩展插件，帮助你自动完成斗鱼鱼吧的每日签到，管理关注的主播鱼吧。

## ✨ 功能特性

### 🎯 核心功能
- **自动签到**：访问鱼吧帖子页面时自动识别签到按钮，完成签到
- **批量签到**：一键批量打开所有已保存的主播鱼吧，自动签到后自动关闭页面
- **关闭签到标签**：一键关闭所有打开的鱼吧签到标签页（所有带有 open_type=auto_check_in 参数的标签）
- **全自动签到**：支持「每天第一次访问斗鱼时自动签到」开关，开启后无需手动操作，每天自动完成所有签到
- **主播管理**：在鱼吧列表页一键保存/移出关注的主播，支持置顶、删除操作
- **快捷导航**：一键直达「我关注的鱼吧」和「全部鱼吧」页面
- **数据同步**：所有数据保存在浏览器本地，永不丢失

### 📱 使用体验
- 无需手动逐个访问每个主播鱼吧
- 支持动态页面自动识别签到
- 支持单页应用路由变化监听，动态添加操作按钮永不消失
- 美观的自定义弹窗，统一的交互体验

## 🚀 安装方法

1. 打开 Chrome 浏览器，进入 `chrome://extensions/`
2. 开启右上角的「开发者模式」
3. 点击左上角的「加载已解压的扩展程序」
4. 选择当前目录即可安装完成

## 📖 使用说明

### 1. 添加关注主播
- 访问「我关注的鱼吧」或「全部鱼吧」页面
- 每个主播卡片右侧会出现「添加」按钮，点击即可保存到本地
- 已添加的主播显示「移出」按钮，点击可取消关注

### 2. 自动签到
#### 手动触发签到
- 点击插件图标，选择「鱼吧自动签到」按钮
- 插件会自动在后台打开所有已保存的主播鱼吧页面
- 每个页面会自动完成签到，3秒后自动关闭页面
- 全程无需手动操作，等待全部签到完成即可

#### 关闭所有签到标签
- 如果需要中途停止签到或者关闭所有打开的签到页面，点击「关闭签到标签」按钮
- 插件会自动关闭所有带有 `open_type=auto_check_in` 参数的标签页，不会影响其他正常打开的斗鱼页面

#### 全自动签到（推荐）
- 打开插件弹窗，开启「每天第一次访问斗鱼时自动签到」开关
- 之后只要你每天第一次打开任意斗鱼网站（包括主站、鱼吧等），插件会自动在后台执行全部签到
- 签到完成后会自动记录，同一天内不会重复触发，完全解放双手

### 3. 管理已保存的主播
- 打开插件 popup 可以看到所有已保存的主播列表
- ⬆️ 置顶按钮：点击将主播移到列表最前面
- ❌ 删除按钮：点击确认后移除该主播
- 点击主播头像/名称可以直接打开对应鱼吧页面

## ⚙️ 权限说明

插件仅申请必要的权限：
- `storage`：保存你关注的主播列表、自动签到开关状态、签到记录到本地
- `activeTab`：获取当前活动标签页信息
- `scripting`：向斗鱼页面注入脚本实现自动操作
- `https://*.douyu.com/*`：仅在斗鱼相关域名下生效，不会访问其他任何网站，安全可靠
